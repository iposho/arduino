# Arduino / ESP32 Projects

Домашние проекты на ESP32 для мониторинга климата и качества воздуха.

## Проекты

### 1. esp32_balcony_pms5003_bme280 — Балконная метеостанция

ESP32 + BME280 + PMS5003 + LCD 1602A + RGB-светофор.

| Датчик | Параметры |
|--------|-----------|
| **BME280** | Температура, влажность, давление |
| **PMS5003** | PM1.0, PM2.5, PM10 (серия замеров со статистикой: min, max, median, mean, stddev) |

**Дисплей LCD 1602A** — 4 страницы с автопереключением каждые 5 сек:
- System — WiFi RSSI, статус BME/PMS
- AQI — PM2.5 + цвет светофора
- Timer — таймер до следующего замера, uptime, буфер
- Reboot — причина последней перезагрузки

**RGB-светофор** — цвет по уровню PM2.5:
- Зелёный: ≤12 мкг/м³
- Жёлтый: ≤35 мкг/м³
- Красный: >35 мкг/м³

**Отправка данных:**
- **Supabase** — климат + пыль каждые 5 минут (медиана из буфера 10 замеров)
- **Telegram** — рапорт при загрузке, алерты при ошибках

**Надёжность:**
- Hardware Watchdog (15 сек)
- Суточный профилактический ребут с уведомлением в Telegram
- Ребут при потере WiFi >10 минут
- Детекция залипшего BME280 (10 одинаковых показаний подряд → алерт + ребут)
- Детекция залипшего PMS5003 (6 одинаковых медиан подряд → алерт + ребут)
- Счётчик ошибок Supabase с алертом после 3 подряд
- Кольцевой лог событий (15) и ошибок (10) в RAM

**Telegram-команды:**
| Команда | Описание |
|---------|----------|
| `/balcony_esp_rst` | Удалённая перезагрузка ESP32 |
| `/balcony_status` | Uptime, RSSI, heap, состояние датчиков, показания |
| `/balcony_logs` | Последние 15 событий (WiFi, Supabase, PMS) |
| `/balcony_errors` | Последние 10 ошибок |

**Пины:**

| Компонент | GPIO |
|-----------|------|
| PMS5003 RX | 16 |
| PMS5003 TX | 17 |
| BME280 I²C | стандарт (адрес 0x76) |
| LCD SDA | 18 |
| LCD SCL | 19 |
| LED R | 25 |
| LED G | 26 |
| LED Y | 27 |

### 2. esp32_flat_bme280 — Комнатный дисплей

ESP32 + BME280 + 1.8" TFT ST7735 + джойстик.

- Показывает температуру, влажность, давление (локальные и с балкона через Supabase)
- AQI-индекс (EPA) на основе PM2.5/PM10 с балкона
- Экраны: Home, Outdoor, AQI, время/дата, системная информация
- Навигация джойстиком (лево/право — экраны, кнопка — часы, вниз — info)
- EMA-фильтр показаний BME280 (α=0.1)
- Отправка данных в Supabase каждые 10 минут

**Пины:**

| Компонент | GPIO |
|-----------|------|
| TFT CS | 5 |
| TFT DC | 2 |
| TFT RST | 4 |
| TFT SCLK | 18 |
| TFT MOSI | 23 |
| BME280 SDA | 21 |
| BME280 SCL | 22 |
| Joystick X | 34 |
| Joystick Y | 35 |
| Joystick SW | 32 |

## Настройка

1. Скопируйте `secrets.example.h` → `secrets.h` в папке нужного проекта
2. Заполните Wi-Fi, Supabase, Telegram данные
3. Откройте `.ino` в Arduino IDE
4. Установите библиотеки:
   - `Adafruit BME280 Library`
   - `Adafruit ST7735 and ST7789 Library` (комнатный дисплей)
   - `BitBang_LiquidCrystal_I2C` (балконный LCD)
   - `PMS Library` (балконный PMS5003)
   - `ArduinoJson` (комнатный дисплей)
